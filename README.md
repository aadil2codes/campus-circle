# 🎓 CircleNet

> 🌍 Connecting Students Beyond Campus

CircleNet is a student networking platform designed to help college students connect, collaborate, and grow with peers from different institutions across India.

Whether you're looking for 🤝 project teammates, 🚀 hackathon partners, 💼 placement guidance, 📚 study resources, or 🎯 career opportunities, CircleNet brings everything together in one place.

🔗Live Preview - https://circle-net-ruby.vercel.app/
---

## ✨ Features

### 🏘️ Communities
- Join communities created by students
- Discover trending discussions
- Explore communities across colleges
- Create your own community after eligibility unlocks

### 📰 Social Feed
- Browse posts from different communities
- ❤️ Like posts
- 💬 Comment on discussions
- 🔖 Save useful content
- 🔄 Share posts with others

### 🤝 Student Networking
- Connect with students across India
- Build meaningful academic and professional relationships
- Discover students with similar interests

### 💬 Messaging
- Direct student-to-student messaging
- Real-time conversations
- Private communication

### 👤 Profiles
- Personalized student profiles
- College, branch, and academic details
- Skills and interests showcase
- Profile customization

### 🛡️ Security & Ownership
- Secure authentication
- Protected user data
- Users can only edit their own content
- Permission-based actions

## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/aadil2codes/circle-net.git
```

Move into the project:

```bash
cd circle-net
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
